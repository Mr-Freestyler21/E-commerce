Koji Cosmetics is a modern, responsive single-page marketing website built for a fictional skincare brand specializing in gentle, oat-milk-based products.

This project focuses on clean design, effective product presentation, and smooth user experience using cutting-edge front-end techniques like smooth scrolling and scroll-based fade-in effects.
